# ⚡ ByteSize STEM — Landing Page

> Step Out of the Textbook. Step Into Real STEM.

Welcome to the official repository for the **ByteSize STEM** landing page. Built entirely from scratch using clean, modern **HTML, CSS (General Sans typography, custom variables), and JavaScript**.

---

## 🚀 About the Project

**ByteSize STEM** is a student-led, non-profit NGO dedicated to rescuing youth from boring memorization. We deliver high-impact, hands-on technology bootcamps directly to schools and provide elite training to help students excel in national and international STEM Olympiads.

As the **Founder**, I built this responsive landing page to showcase our core mission, map out our learning modules, highlight our community reach, and introduce our founding team.

### ✨ Key Features
* 🎨 **Neo-brutalist Aesthetic:** Vibrant color palette using strict custom CSS variables (`--lime`, `--forest`, `--blue`, `--maroon`).
* 📱 **Fully Responsive Layout:** Designed using modern CSS Grid and Flexbox to look flawless on any screen size.
* 🃏 **Interactive Card Deck:** Custom JavaScript-driven rotating slider to highlight active school programs in real-time.
* ⚡ **Fluid Performance:** Optimized for fast load times and clean element reveal transitions.
